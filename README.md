# ♻️ PlasticToPoints

**Nepal's Leading Plastic Recycling Credits Platform**

PlasticToPoints is a high-end, minimalist environmental tech platform designed to revolutionize waste management in Nepal. Developed by **Team TerraSync**, our mission is to turn plastic waste into value through a smart reward-based ecosystem.

---

## 🚀 VISION
**Problem -> Tech Solution -> Impact**
- **Problem**: Nepal faces a growing plastic waste crisis with inefficient recycling channels.
- **Tech Solution**: A smart platform that tracks plastic collection and rewards users with points that can be redeemed for items or credit.
- **Impact**: Cleaner streets, empowered communities, and a circular economy starting from Janakpur.

## 🛠️ TECH STACK
- **Frontend**: React 18 + Vite (TypeScript)
- **Styling**: Tailwind CSS 4.0
- **Database/Auth**: Supabase (PostgreSQL)
- **Animations**: Motion (formerly Framer Motion)
- **Icons**: Lucide React
- **SEO/AEO**: React Helmet Async + JSON-LD Structured Data

## ✨ KEY FEATURES
- **User Dashboard**: Track your recycling impact and points balance.
- **Smart Scan Tracking**: Register plastic collection entries (Hub-based).
- **Redemption Store**: Swap points for real-world rewards.
- **Admin Portal**: Manage users, approve scans, and fulfill redemptions.
- **SEO Optimized**: Fully prepared for Search Engines and AI Answer Engines (Perplexity, Gemini, etc.).

---

## 📦 PROJECT STRUCTURE
- `src/pages`: Main application views (Home, About, Admin, etc.).
- `src/components`: Reusable UI components and Layout.
- `src/lib/supabase.ts`: Backend integration configuration.
- `firebase-blueprint.json`: (Legacy/Reference) Data structure mapping.

## 🛠️ LOCAL DEVELOPMENT

1. **Clone the repository**:
   ```bash
   git clone <your-repo-url>
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Set up Environment Variables**:
   Create a `.env` file based on `.env.example`:
   ```env
   VITE_SUPABASE_URL=your_supabase_url
   VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
   ```

4. **Run the development server**:
   ```bash
   npm run dev
   ```
